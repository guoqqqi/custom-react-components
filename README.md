# custom-react-components

## Uses

```shell
# npm
npm install @guoqqqi/custom-react-components@0.0.3

# yarn
yarn add @guoqqqi/custom-react-components@0.0.3
```

```tsx
...
import { Button, Card } from '@guoqqqi/custom-react-components';
...

return (
  <>
    <Button onClick={() => ()} label='Hello Word' />
    <Card
      imgvisible
      title='My Blog'
      desc='Learn all about My storyies, technical know-how and tips.'
      buttonHref='href'
    />
  </>
)
```
